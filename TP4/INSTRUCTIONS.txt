1) DANS PGADMIN, CRÉER UNE BASE DE DONNÉES NOMMÉE "neflixbd" en minuscules.
2) DANS PGADMIN, CLIQUE DROIT SUR "netflixbd" ET CLIQUER SUR "Query Tool".
3) EFFECTUER LA REQUÊTE SUIVANTE DANS LE "Query Tool": "CREATE USER admin encrypted password 'admin';
GRANT ALL PRIVILEGES ON DATABASE netflixdb to admin;".
4) OUVRIR UN TERMINAL DANS LE DOSSIER "server" qui existe dans le dossier "TP4".
5) EXECUTER LA COMMANDE "npm start" DANS LE DOSSIER "server".
6) OUVRIR UN TERMINAL DANS LE DOSSIER "client" qui existe dans le dossier "TP4".
7) EXECUTER LA COMMANDE "npm start" DANS LE DOSSIER "client".
8) VOTRE BROWSER DEVRAIT S'OUVRIR ET AFFICHER L'APPLICATION, SI CE N'EST PAS LE CAS, NAVIGUEZ VERS L'ADRESSE : "http://localhost:4200/home" DANS VOTRE FURETEUR.
9) CLIQUEZ SUR "Initialiser la base de données"
10) POUR VOUS CONNECTER EN TANT QU'ADMINISTRATEUR, UTILISEZ LES CREDENTIALS SUIVANTS:
        Adresse courriel: admin@admin.ca
        Mot de passe: admin
